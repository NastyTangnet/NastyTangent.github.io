Drop your denomination cover images in this folder so the homepage cards can use consistent "BU example" icons.

Expected filenames (PNG/JPG/WEBP all work, but PNG is recommended):
- `dollar.png`
- `penny.png`
- `commemoratives.png`
- `nickel.png`
- `dime.png`
- `quarter.png`
- `half_dollar.png`

If a cover is missing, the site falls back to using one of your own coin photos for that section.
